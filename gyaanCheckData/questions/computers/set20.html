Microsoft Natural Keyboard Shortcuts

  • Log off WindowsWindows Logo + L

  • Starts Print ManagerWindows Logo + P

  • Opens Control PanelWindows Logo + C

  • Starts ClipboardWindows Logo + V

  • Opens Keyboard Properties dialog boxWindows Logo + K

  • Opens Mouse Properties dialog boxWindows Logo + I

  • Starts Accessibility Options (if installed)Windows Logo + A

  • Displays the list of Microsoft IntelliType shortcut keyWindows Logo + SPACEBAR

  • Toggles CAPS LOCK on and offWindows Logo+S
