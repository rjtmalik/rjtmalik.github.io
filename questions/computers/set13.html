Microsoft Excel Shortcuts - 2

  • Switch between open workbooks or windows.Ctrl + F6

  • Move between work sheets in the same document.Ctrl + Page up

  • Move between Two or more open Excel files.Ctrl + Tab

  • Create a formula to sum all of the above cells.Alt + =

  • Select entire column.Ctrl + Space

  • Format number in comma format.Ctrl + Shift + 1

  • Format number in currency format.Ctrl + Shift + 4

  • Move to cell A1.Ctrl + Home

  • Delete the selected column or row.Ctrl + -
