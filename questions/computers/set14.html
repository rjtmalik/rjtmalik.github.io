Microsoft Word Shortcuts - 1

  • Close the active window / document.Ctrl + W

  • Undo an action.Ctrl + Z

  • Redo the last action or repeat an action.Ctrl + Y

  • Save a document.Ctrl + S

  • Print a document.Ctrl + P

  • Insert a hyperlink.Ctrl + K

  • Arrow Go back one page.Alt + Left

  • Arrow Go forward one page.Alt + Right

  • Copy selected text or graphics to the Office Clipboard.Ctrl + C
